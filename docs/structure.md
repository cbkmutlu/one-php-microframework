##ONE PHP Micro Framework Documentation

This is the directory structure you will end up with following the instructions in the Installation Guide.
###Regular
    |-- ROOT
    |       |-- .htacess
    |       |-- index.php
    |       |-- one_framework.php
    |       |-- views/
    |       |-- assets/
    
or
####MVC

    |-- ROOT
    |       |-- .htacess
    |       |-- index.php
    |       |-- src/
    |       |--     |-- controllers/    
    |       |--     |-- one_framework.php
    |       |--     |-- views/
    |       |-- assets/
